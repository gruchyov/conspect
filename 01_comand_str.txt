Редакция
0 git --version - версия установленной гит
1 git config --global user.name "........"
2 git config --global user.email "........"
3 git config --list - проверить настройки

4 git config --unset user.name -  удалить настройку

5 git config --global core.editor "code --wait" - установить редактор VSCode
6 сохранить комментарий в коммите -  Ctrl+s
7 выйти в терминал - Ctrl+w, enter




0 ls -la - смотреть содержимое папки, выводятся все файлы и папки, которые 
           выводятся в текущей папке

1 git status - есть ли в папке репозиторий, когда от создан - проверка, 
               есть ли в репозитории несохраненные изменения.

2 git init - создание репозитория в папке проекта

3 git add . - добавление файла в репозиторий

4 git commit -m "комментарий к изменению"

5 git show - убедиться, что изменения сохранились в репозитории и просмотреть
             детали о последнем коммите, хэш и комментарий коммита, автора 
             коммита и информация, какие конкретно изменения были сохранены.
               git show показывает информацию о последнем коммите и его содержимом.
             Если нажимать ENTER можно просмотреть все файлы, которые в рамках этого
             коммита были внесены в репозиторий.
  git show --pretty=fuller -с указанием того, кто коммитил
               (Нажимал - не видно пока, может не туда смотрел)  

6 git log - просмотреть всю историю изменений, все коммиты которые были в 
            репозитории, их хэши, дату и время и комментарии.

   какое то вечернее изменение №1
   утреннее № 2
   дневное № 3
